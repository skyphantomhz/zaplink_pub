如何在 macOS 和 Windows 上安装 zapLink

zapLink 完全免费开放使用，但由于它并非通过 Mac App Store 或 Microsoft Store 发行，macOS 和 Windows 首次打开时可能会显示安全警告。这对于独立发行的免费软件是正常现象——以下是解除警告的具体步骤，不到一分钟即可完成。

🍎 macOS：“Apple 无法验证...是否包含恶意软件”

macOS 的 Gatekeeper 机制默认会拦截来自未识别开发者的应用。要允许运行 zapLink：

方法一 — 右键打开（最快）

  1. 在“下载”文件夹中找到 zapLink-macos.dmg 并打开，然后将 zapLink 拖入“应用程序”文件夹。
  2. 打开“应用程序”文件夹，找到 zapLink。
  3. 右键点击（或按住 Control 点击）zapLink 图标，从菜单中选择打开。
  4. 系统会弹出关于未知开发者的警告，再次点击打开确认。
  5. zapLink 即可启动，macOS 会记住此次选择，以后打开无需重复此步骤。

方法二 — 通过系统设置允许

  1. 尝试正常双击打开 zapLink，macOS 会拦截并显示警告。
  2. 打开系统设置 → 隐私与安全性。
  3. 向下滚动到安全性部分，会看到 zapLink 被阻止运行的提示。
  4. 点击仍要打开。
  5. 如有提示，请再次确认（可能需要输入 Mac 密码或使用 Touch ID）。
在较旧版本的 macOS（Catalina/Big Sur）上，该设置可能位于系统偏好设置 → 安全性与隐私 → 通用中。

🪟 Windows：“Windows 已保护你的电脑”

Windows SmartScreen 会对尚未建立下载信誉的应用显示此警告。要允许运行 zapLink：

  1. 正常运行 zapLink-windows-setup.exe。
  2. 当出现蓝色的“Windows 已保护你的电脑”界面时，点击更多信息。
  3. 此时会出现仍要运行选项，点击它。
  4. 安装程序将继续正常运行，按照安装步骤完成 zapLink 的安装。
如果你的防病毒软件也拦截了安装程序，只要你是从官方 GitHub Releases 页面下载的，就可以放心允许运行。

常见问题

为什么 macOS 提示 zapLink 来自身份不明的开发者？

zapLink 是一款免费、独立发行的应用，尚未加入苹果的付费公证计划。这个警告并不代表应用不安全，只是尚未经过苹果的付费审核流程。

为什么 Windows SmartScreen 会拦截 zapLink？

SmartScreen 会对尚未建立良好声誉或没有付费代码签名证书的应用发出警告。你可以点击“更多信息”然后“仍要运行”来安全运行该应用。

点击“仍要打开”或“仍要运行”安全吗？

只要你是从官方 GitHub Releases 页面下载的 zapLink，这样操作就是安全的。请仅对你信任来源的软件执行此操作。
