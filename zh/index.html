zapLink — 将文件发送到智能电视

zapLink 让你通过 Wi-Fi 或局域网把照片、视频和文件即时发送到智能电视。它将你的电脑变成一个私有本地服务器：选择一个文件夹，点击“开始”，你的智能电视——或同一网络下的任何手机——扫描二维码即可在浏览器中直接打开。无需云端，无需数据线，无需安装应用。完全免费，支持 macOS、Windows 和 Linux。

如何将文件发送到智能电视

  1. 在电脑上选择一个想要共享的文件夹。
  2. 点击“开始”——本地服务器会立即启动，并生成一个新的 PIN 码。
  3. 打开智能电视的浏览器，扫描二维码，输入 PIN 码，文件就会直接在电视上打开。

安全性

每位访问者都需要输入 4 位 PIN 码才能连接，你可以实时查看并撤销任何已连接的设备。

常见问题

如何将文件发送到智能电视？

在电脑上打开 zapLink，选择一个文件夹，点击开始。二维码和 PIN 码会出现——打开智能电视的浏览器，扫描二维码（或输入地址），输入 PIN 码，文件就会直接在电视上打开。电视端无需安装任何应用。

智能电视需要安装应用吗？

不需要。任何带浏览器的智能电视——三星、LG、Android TV 等——都可以打开你共享的文件夹。无需应用商店，无需安装。

我的数据会上传到云端吗？

不会。zapLink 直接通过你的本地 Wi-Fi 或局域网传输文件，不会上传到任何服务器——文件不会离开你的网络。

需要联网才能使用吗？

不需要。只要电脑和智能电视连接同一个 Wi-Fi 或局域网，zapLink 就可以完全离线使用。

连接是如何保证安全的？

每个会话都通过 4 位 PIN 码保护，你可以实时查看并撤销任何已连接的设备。

支持哪些操作系统？

zapLink 支持 macOS 12+、Windows 10+ 以及 Ubuntu 20.04+（及兼容的 Linux 发行版）。
