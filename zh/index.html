zapLink — 通过 Wi-Fi 和局域网（LAN）传输文件

zapLink 让你通过 Wi-Fi 或局域网即时传输文件。它将你的电脑变成一个私有本地服务器：选择一个文件夹，点击“开始”，同一 Wi-Fi 或局域网下的任何手机、平板或智能电视都能立即在浏览器中打开它——无需云端，无需数据线。完全免费，支持 macOS、Windows 和 Linux。

使用方法

  1. 在电脑上选择一个想要共享的文件夹。
  2. 点击“开始”——本地服务器会立即启动，并生成一个新的 PIN 码。
  3. 同一网络下的设备扫描二维码、输入 PIN 码即可连接，实现 Wi-Fi/局域网文件传输。

安全性

每位访问者都需要输入 4 位 PIN 码才能连接，你可以实时查看并撤销任何已连接的设备。

常见问题

我的数据会上传到云端吗？

不会。zapLink 直接通过你的本地 Wi-Fi 或局域网传输文件，不会上传到任何服务器——文件不会离开你的网络。

需要联网才能使用吗？

不需要。只要设备连接同一个 Wi-Fi 或局域网，zapLink 就可以完全离线使用。

连接是如何保证安全的？

每个会话都通过 4 位 PIN 码保护，你可以实时查看并撤销任何已连接的设备。

支持哪些操作系统？

zapLink 支持 macOS 12+、Windows 10+ 以及 Ubuntu 20.04+（及兼容的 Linux 发行版）。

手机或电视需要安装应用吗？

不需要。同一网络下的任何手机、平板或智能电视只需使用浏览器即可连接，无需安装任何应用。
