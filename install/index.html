How to Install zapLink on macOS & Windows

zapLink is free and open for anyone to use, but because it's distributed outside the Mac App Store and Microsoft Store, macOS and Windows may show a security warning the first time you open it. This is normal for independent, free software — here's exactly how to get past it in under a minute.

🍎 macOS: "Apple could not verify... is free of malware"

macOS Gatekeeper blocks apps from developers it doesn't recognize by default. To allow zapLink:

Option A — Right-click to open (fastest)

  1. Locate zapLink-macos.dmg in your Downloads folder and open it, then drag zapLink into Applications.
  2. Open your Applications folder, find zapLink.
  3. Right-click (or Control-click) the zapLink app icon and choose Open from the menu.
  4. A dialog will appear warning about an unidentified developer. Click Open again to confirm.
  5. zapLink will now launch, and macOS will remember your choice for future launches.

Option B — Allow via System Settings

  1. Try opening zapLink normally (double-click). macOS will block it and show a warning.
  2. Open System Settings → Privacy & Security.
  3. Scroll down to the Security section — you'll see a message that zapLink was blocked.
  4. Click Open Anyway.
  5. Confirm again if prompted (you may need to enter your Mac password or Touch ID).
On older macOS versions (Catalina/Big Sur), this setting may be under System Preferences → Security & Privacy → General instead.

🪟 Windows: "Windows protected your PC"

Windows SmartScreen shows this warning for apps that don't yet have an established download reputation. To allow zapLink:

  1. Run zapLink-windows-setup.exe as usual.
  2. When the blue "Windows protected your PC" screen appears, click More info.
  3. A new option, Run anyway, will appear — click it.
  4. The installer will continue normally. Follow the setup steps to finish installing zapLink.
If your antivirus software also flags the installer, you can safely allow it — just make sure you downloaded it from the official GitHub Releases page.

Frequently asked questions

Why does macOS say zapLink is from an unidentified developer?

zapLink is a free, independently distributed app and isn't yet enrolled in Apple's paid notarization program. This warning doesn't mean the app is unsafe — it just hasn't gone through Apple's paid review process.

Why does Windows SmartScreen block zapLink?

SmartScreen warns about apps that don't yet have an established reputation or a paid code-signing certificate. You can still run it safely by clicking "More info" then "Run anyway".

Is it safe to click "Open Anyway" or "Run anyway"?

Yes — as long as you downloaded zapLink from the official GitHub Releases page. Only bypass these warnings for software from a source you trust.
