Cómo Instalar zapLink en macOS y Windows

zapLink es gratuito y está abierto para que cualquiera lo use, pero como se distribuye fuera de la Mac App Store y la Microsoft Store, macOS y Windows pueden mostrar un aviso de seguridad la primera vez que lo abras. Esto es normal para software gratuito e independiente — aquí te explicamos exactamente cómo superarlo en menos de un minuto.

🍎 macOS: "Apple no pudo verificar... esté libre de malware"

Gatekeeper de macOS bloquea por defecto las apps de desarrolladores que no reconoce. Para permitir zapLink:

Opción A — Clic derecho para abrir (más rápido)

  1. Localiza zapLink-macos.dmg en tu carpeta de Descargas y ábrelo, luego arrastra zapLink a Aplicaciones.
  2. Abre tu carpeta de Aplicaciones, busca zapLink.
  3. Haz clic derecho (o Control-clic) en el icono de zapLink y elige Abrir en el menú.
  4. Aparecerá un aviso sobre un desarrollador no identificado. Haz clic en Abrir otra vez para confirmar.
  5. zapLink se abrirá, y macOS recordará tu elección para futuras aperturas.

Opción B — Permitir desde Configuración del Sistema

  1. Intenta abrir zapLink normalmente (doble clic). macOS lo bloqueará y mostrará un aviso.
  2. Abre Configuración del Sistema → Privacidad y Seguridad.
  3. Desplázate hasta la sección de Seguridad — verás un mensaje de que zapLink fue bloqueado.
  4. Haz clic en Abrir de todos modos.
  5. Confirma de nuevo si se solicita (puede requerir tu contraseña de Mac o Touch ID).
En versiones anteriores de macOS (Catalina/Big Sur), esta opción puede estar en Preferencias del Sistema → Seguridad y Privacidad → General.

🪟 Windows: "Windows protegió tu PC"

Windows SmartScreen muestra este aviso para apps que aún no tienen una reputación de descarga establecida. Para permitir zapLink:

  1. Ejecuta zapLink-windows-setup.exe normalmente.
  2. Cuando aparezca la pantalla azul "Windows protegió tu PC", haz clic en Más información.
  3. Aparecerá una nueva opción, Ejecutar de todas formas — haz clic en ella.
  4. El instalador continuará normalmente. Sigue los pasos para completar la instalación de zapLink.
Si tu antivirus también marca el instalador, puedes permitirlo con seguridad — solo asegúrate de haberlo descargado desde la página oficial de GitHub Releases.

Preguntas frecuentes

¿Por qué macOS dice que zapLink es de un desarrollador no identificado?

zapLink es una app gratuita e independiente que aún no está inscrita en el programa de notarización de pago de Apple. Este aviso no significa que la app sea insegura — solo que no ha pasado por el proceso de revisión de pago de Apple.

¿Por qué Windows SmartScreen bloquea zapLink?

SmartScreen advierte sobre apps que aún no tienen una reputación establecida o un certificado de firma de código de pago. Puedes ejecutarla de forma segura haciendo clic en "Más información" y luego en "Ejecutar de todas formas".

¿Es seguro hacer clic en "Abrir de todos modos" o "Ejecutar de todas formas"?

Sí, siempre que hayas descargado zapLink desde la página oficial de GitHub Releases. Solo omite estos avisos para software de una fuente en la que confíes.
